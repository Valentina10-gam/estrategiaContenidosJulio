# TKC | Estrategia de Contenido Orgánico · Julio 2026

Landing/dashboard local en HTML, CSS y JavaScript para presentar la estrategia orgánica de julio 2026 de TKC Fumigaciones Group.

## Cómo abrir

1. Abre `index.html` directamente en tu navegador.
2. No requiere backend, instalación ni servidor local.
3. Para editar la parrilla mensual, modifica los objetos dentro de `data.js`.

## Archivos

- `index.html`: estructura de la experiencia.
- `styles.css`: identidad visual, responsive y animaciones.
- `script.js`: filtros, buscador, modal, navegación y progreso de lectura.
- `data.js`: datos editables de las 20 piezas estratégicas.
